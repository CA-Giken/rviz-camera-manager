# Rvizカメラ操作・視点登録

# Installation 
```
cd ~/catkin_ws/src
git clone https://github.com/CA-Giken/rviz_camera_manager.git

python3 -m pip install pysimplegui

cd ~/catkin_ws
catkin build rviz_camera_manager
```

# Run
```
roscore
```

```
source ~/catkin_ws/devel/setup.bash
roslaunch rviz_camera_manager main.launch
```
