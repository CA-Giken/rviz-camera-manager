# Rvizカメラ操作・視点登録

# Installation 
```
cd ~/catkin_ws/src
git clone https://github.com/CA-Giken/rviz-camera-manager.git

python3 -m pip install pysimplegui

cd ~/catkin_ws
catkin build rviz-camera-manager
```

# Run
```
roscore
```

```
source ~/catkin_ws/devel/setup.bash
rosrun rviz-camera-manager main.py
```
